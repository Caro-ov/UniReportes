Cómo Crear un Reporte

Sigue estos sencillos pasos para reportar un problema de infraestructura en el campus:

1

Selecciona la ubicación

Usa el mapa interactivo para indicar dónde se encuentra el problema.

2

Describe el problema

Añade detalles y fotos para que podamos entender mejor la situación.

3

Envía el reporte

Revisa la información y envía tu reporte. Recibirás una notificación.
